# Архітектура UXT Pipeline

## 1. Загальний огляд
UXT Pipeline — модульна система для обробки неструктурованих документів, семантичного чанкінгу та пошуку на основі ембеддингів.

Система складається з трьох рівнів:
1. **Ingest Layer** — парсинг PDF/DOCX/HTML.
2. **Transformation Layer** — нормалізація, чанкінг, NLP-обробка.
3. **Indexing & Retrieval Layer** — ембеддинги, індексація, семантичний пошук.

Додатково:
- **API Layer (FastAPI)** — REST інтерфейси.
- **UI Layer (Streamlit)** — інтерактивний фронтенд.
- **CLI Layer** — автоматизація та інтеграція.
